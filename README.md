# Student and classes

This is a school assignment with the purpose making of CRUD operations on a local json-server while focusing on using React's **useState** and **useEffect** as well as the concepts of **lifting state up** and **controlling components**

## Tech Stack

- **React**
- **Json-server**
- **JavaScript**
- **CSS**
- **Vite**
- **Bootstrap**

## How to run

- Make sure you have `Node.js` on your machine
- Clone this project
- Navigate to the root folder of this project
- Run command `npm install`
- Run command `npm run dev`
- Run command `npm run jsonserver`
- Communicate with the API by visiting `http://localhost:5173`
